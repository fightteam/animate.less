# Animate.less

基于css3的常用动画，根据animate改用less，为了方便less用户的使用。

## System Requirement
* [nodejs](http://nodejs.org/)
* [grunt](http://gruntjs.com/)

## Getting Started

####安装nodejs
```
$ npm install
```
####安装grunt
####编辑animate.less
####运行spa-run.bat
